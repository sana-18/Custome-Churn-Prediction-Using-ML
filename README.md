# Customer-Churn-Prediction-Using-ML
# Overview 
This repository contains the code and resources for a machine learning project focused on predicting customer churn in a subscription-based telecommunication business.
Customer churn, or the loss of customers, is a critical metric for businesses, and predicting it can help in implementing proactive strategies to retain valuable customers.
This project was done as part of requirements of ML module at the higher school of technology of meknès.
# Objective
The primary goal of this project is to develop an accurate machine learning model that predicts the likelihood of a customer churning based on historical data. We aim to leverage the power of XGBoost, a high-performance gradient boosting algorithm, to achieve robust predictive performance.
# Additional
Dataset: Utilizing a curated dataset containing historical customer interactions, subscription details, and churn labels.

Data Preprocessing: Cleaning and preparing the data for model training.

Model Training: Implementing XGBoost as the primary algorithm for its excellent performance in classification tasks.

Evaluation: Assessing the model's accuracy, precision, recall, and F1-score to measure its effectiveness.

Deployment: Making a streamlit app resuming all the steps of the project.

Documentation: Providing detailed documentation on data sources, methodology for the whole project process.
